DNS



DNS is the naming system for computers that relates IP address with Domain Names.

An DNS server is like the address book. We look up a person’s name or business name to find out their location or phone number. In the same sense, we specify the domain we would like to reach, and the DNS gives the computer the IP address that correlates with the place.

Following this example of house, ICANN can be recognized as the local government that manages who owns a piece of land, and other details such of a property.

DNS is the “address book” of the internet. The internet would be possible with DNS. There are four parts to DNS servers. DNS recursor, Root NameServer, TLD name server, and Authoritative name server.

ICANN works by managing DNS and IP addresses. Millions of domain names are managed by ICANN all over the world. This is a non-profit organization that works with governments all around the world inorder to manage the internet. Although, the content of a website is not managed by ICANN, there have been many disputes regarding generic top-level domains such as “.amazon” and “.sucks”. THe first step is the Recursive name server. This is operated by the ISP provider. If the address is not found, it moves to look the root servers which manages the top level domains. This is then sent to the appropriate top level domain name server. Then Authoritative name servers are contacted which contains authoritative list of IP address and matching URLs. This list is updated whenever a domain is purchased or sold. For the future, your computer will cache DNS entries in your own computer or the recursive error.
