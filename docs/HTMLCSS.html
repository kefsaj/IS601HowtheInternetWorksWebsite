TW3C and HTML and CSS



W3C is the World wide web Consortium. W3C develops the standard of the World Wide Web.
HTML is hyper text markup language. Regarded as the standard markup language, html works by giving the structural foundation of a website.
CSS is the Cascading Style sheets. This is called the presentation layer in a browser that works along html to make human friendly webpages.

W3C is like the authoritative figure that make sure that everything works together sharing common principles. This is like when drivers have to follow rules and regulations that are set by the government.
HTML is like the secatuary who works at the office that makes the boss’s life easier, just like how html makes the user’s life easier in deciphering the information.
CSS is like the decorator that makes everything beautiful and elegant for the people who are the user.

The W3C educated others about the internet regarding developing software and creating a better internet environment. It was created in 1994 in MIT, to have a common set of principles and components picked by the W3C.
HTML is any text that be shown on a computer screen that contain link to other test. HTML is a markup language by giving meaning and structure to text that is shown to the user. Components are divided into tags. HTML is understood by all browsers, including phones tablets and computers.
CSS plays a huge part in the world wide web along with HTML and Javascript. CSS works with all kinds of browser to make content elegant and readable.
