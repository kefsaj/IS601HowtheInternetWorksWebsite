HTTP and HTTPS protocols

HTTP and HTTPS protocols



HTTP is Hypertext Transfer Protocol.

HTTPS is Hypertext Transfer Protocol Secure, extension of the HTTP.

HTTP is special because unlike a regular text, hypertext links to other via hyper links. It is regarded as the backbone of the internet. HTTP follows the Client/Server model, serving what the client wants divided by HTTP Requests and HTTP Responses. These requests typically contain the HTTP version type, URL, HTTP method, request headers and the body. The response is consists of the HTTP status code, response headers and sometimes the HTTP body.

The S at the end of HTTPS is stands for Secure. HTTPS is an extension of HTTP to enable secure communication across a computer network and is broadly used on the internet. HTTPS = HTTP + cryptographic protocols
