Domain names and ICANN



Domain Name is a human friendly address which points to a website.

ICANN is the Internet corporation for assigned names and numbers.

An example for Domain Name would be the address to your house correlates to how the domain name works. A domain name is like a favorite button to your house and the DNS server converts your favorite button to the latitude and longitude degrees for the GPS program to calculate the address location

Following this example of house, ICANN can be recognized as the local government that manages who owns a piece of land, and other details such of a property.

As websites are accessed IP address, so it is easy for a computer to work with but not for a human. A domain name makes it easy for humans to remember. A Domain Name System Server is used to convert the domain name into the appointee IP address.

ICANN works by managing DNS and IP addresses. Millions of domain names are managed by ICANN all over the world. This is a non-profit organization that works with governments all around the world inorder to manage the internet. Although, the content of a website is not managed by ICANN, there have been many disputes regarding generic top-level domains such as “.amazon” and “.sucks”. In terms of a website, the ICANN in involved in three main ways. ICANN serves itself as a central repository for IP addresses, manages the registry of domain registers and acts as the root server. ICANN acting as the root server guiding where requests to proper Top Level Domains.
